How to publish

1

Copy the JSON entry above using the Copy button.

2

Go to your GitHub repo and open projects.json. Click the pencil icon to edit.

3

Find the last } before the closing ]. Add a comma after it, then paste your new entry.

4

Click Commit changes. Your project will appear on the site within ~1 minute.

To add a photo: Go to your repo → assets/projects/ → click Upload files → upload your image. Use the filename you entered in the form above.
